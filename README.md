# Git Remote Repository Setup (Without GitHub)

## Overview
This project demonstrates setting up a remote Git repository without using GitHub and simulating collaboration between multiple developers.

## Tech Stack
- Git
- Linux
- SSH

## Architecture Workflow
1. Bare repository created on remote server.
2. Developer pushes initial code.
3. Second developer clones repository.
4. Changes are committed and pushed back.

## Key Features
- Remote repository management
- Multi-user collaboration
- Version control operations

## Challenges
- SSH configuration
- Remote connectivity setup
- Managing Git workflows

## Results
- Successful simulation of team collaboration
- Efficient version control handling
